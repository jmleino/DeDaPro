Estimating Car's MPG

Mickey Mouse
27.9.2015

Data Product for MPG Estimation

  • Uses mtcars dataset
  • Linear regression allows accurate mpg estimates
  • Easy to use

User Input

  • Car's horsepower
  • Car's weigth
  • Transmission type

Regression model

The linear regression model used:

lm( mpg ~ wt + hp + am, data = mtcars )


Call:
lm(formula = mpg ~ wt + hp + am, data = mtcars)

Coefficients:
(Intercept)           wt           hp           am  
   34.00288     -2.87858     -0.03748      2.08371

Output

The product dislays the estimated MPG as well as two graphs illustrating the values compared to other car models.
